## 🛠 Installation

Clonez le projet et installez les dépendances pour commencer à l'utiliser :

```bash
npm install
```
🔨 Compilation du SASS
Compilez les styles avec la commande suivante :

```bash
npm run sass
```
🖥 Utilisation :
Lancez index.html dans votre navigateur pour voir le portfolio en action.
